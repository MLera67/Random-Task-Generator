# Random Task Generator

Автор: Виктория Гайфуллина

## Описание
Это простое GUI-приложение для случайного выбора задач с возможностью фильтрации по типу. Также оно сохраняет историю сгенерированных задач в JSON.

## Использование
- Нажмите кнопку "Сгенерировать задачу" для получения случайной задачи.
- Используйте выпадающий список для фильтрации задач по типу (учёба, спорт, работа, все).
- В разделе "Добавить новую задачу" введите задачу и её тип, затем нажмите "Добавить задачу".

## Тесты
Попробуйте добавить новые задачи, выбрать разные фильтры и сгенерировать задачи.
